航海日志｜个人成长记录网页

1.项目简介

航海日志是一个以“航海探索”为主题的个人成长记录网页。
它把日常记录、目标管理、愿望地图、情绪补给和个人资料整合在一起，用更轻松、有故事感的方式记录自己的成长过程。

在这个网页里，每一次记录都像一次出航，每一个目标都像一座等待抵达的岛屿。
我希望它不只是一个普通的待办工具，而是一个能陪伴用户整理生活、记录状态、慢慢成长的个人空间。

2.核心功能

航海日志：记录每日想法、生活状态、学习进度和阶段复盘。
主线任务：管理当前最重要的目标，把长期计划拆成更清晰的行动。
愿望地图：保存想完成的愿望、想抵达的地方和未来想实现的目标。
情绪补给：记录书籍、电影、音乐、灵感等能给自己带来能量的内容。
个人资料：展示用户的昵称、状态、成长方向和阶段目标。

3.设计理念

这个项目将个人成长比作一次航行：

* 日志是每天的航海记录
* 目标是远方的岛屿
* 愿望地图是未来的方向
* 情绪补给是继续前进的能量
* 主线任务是正在行驶的航线

通过这样的设定，网页不仅有记录功能，也更有故事感和陪伴感。

4.项目意义

制作这个网页，是想把“个人成长”这件事变得更具体、更轻松，也更有仪式感。
成长不是一条直线，而是一段不断出发、调整方向、靠近自己的旅程。

“航海日志”希望成为这样一个小小的记录空间：
让用户能把目标写成航线，把愿望放进地图，把情绪当作天气，把每天的行动都看作一次新的出航。
